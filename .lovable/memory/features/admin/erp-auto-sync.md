- إجراء `auto_sync_full` في `erp-sync-outbound`: يزامن السعر القطاعي + الجملة + الرصيد للأصناف `is_active=true` فقط (Whitelist).
- يكتشف أصناف الفيصل التي رصيدها > حد قابل للضبط (افتراضي 10) وغير موجودة في الكتالوج، ويُدخلها كـ `toyota_genuine` نشطة، ثم يُنشئ إشعار لكل أدمن.
- يعمل عبر `pg_cron` job باسم `erp-auto-sync-hourly` كل ساعة (`0 * * * *`).
- زر "تشغيل الآن" + حقل ضبط الحد + تقرير تفصيلي في AdminERPSync (تبويب actions).
