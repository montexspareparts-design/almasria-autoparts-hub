## نظرة عامة
بعد ما الموظف يقفل طلب الدعم (status = `resolved` أو `closed`)، أول مرة العميل يفتح الشات بوت بعدها بيشوف بطاقة تقييم اختيارية فوق الرسائل:

> "كيف كانت تجربتك مع أحمد؟ ⭐⭐⭐⭐⭐"

## التفاصيل التقنية

### قاعدة البيانات
- **جدول `support_request_ratings`**:
  - `support_request_id` (UNIQUE — تقييم واحد لكل طلب)
  - `staff_user_id` — الموظف اللي تم تقييمه
  - `customer_user_id` — العميل اللي قيّم
  - `rating` (1-5)
  - `comment` (اختياري، حتى 300 حرف)

### RLS
- العميل يقدر يدخل تقييم لطلباته المحلولة فقط
- الموظفين (admin + moderator) يقدروا يشوفوا كل التقييمات
- الأدمن يقدر يحذف/يعدّل

### Trigger
- `trg_notify_staff_on_rating` — يبعت إشعار للموظف فور وصول تقييم جديد بعدد النجوم والتعليق

## UX
- **اختياري بالكامل** — زر "تخطي" في الكورنر + زر "تخطي" في الفوتر
- التخطي يتسجل في `localStorage` تحت `rating_skipped_requests` فعدم الإزعاج
- بمجرد اختيار النجوم تظهر خانة التعليق + زر الإرسال
- التعليق placeholder بيتغير حسب التقييم:
  - 4-5 نجوم: "إيه اللي عجبك؟"
  - 1-3 نجوم: "كيف نقدر نتحسن؟"

- `src/components/AIChatBot.tsx` — يستدعيها فوق الرسائل لما الشات يفتح
