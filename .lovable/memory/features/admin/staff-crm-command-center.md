صفحة `/admin?section=daily-dashboard` تعرض `StaffDailyDashboard` لكل من الأدمن والموظف بنفس المحتوى:
- 4 تابات: urgent SLA / search-no-buy / yesterday-visitors / leaderboard
- جداول داعمة: customer_sessions + staff_contact_marks
- أي تحديث يظهر تلقائياً للطرفين (مفيش فرع منفصل للموظف بعد الآن).
